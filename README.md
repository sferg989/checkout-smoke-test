maintainer facing readme

## Local Dev

### Installation

1. `cp .env.example .env`
    - Update variables with values from another project developer
2. `npm i`
3. `npm build-watch` | `npm run build`
4. `npm run once`

### Code Quality Checks

- Linting - `npm run lint`
- Tests - `npm run test` | `npm run test-once`
